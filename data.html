Dataset Overview

The project includes cleaned COVID-19 death data in CSV format. This file contains country-level metrics, dates, population values, and death totals used for analysis.

How to use it

  • Import the CSV into your SQL environment or data analysis tool.
  • Run the SQL script in the repository to explore trends and calculations.
  • Use the dataset to compute infection rates, death rates, and totals per country.
