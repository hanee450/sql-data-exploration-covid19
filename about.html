Project Overview

This site showcases a COVID-19 exploratory data analysis project using SQL. It summarizes the data sources, business questions, and key insights from the repository.

The analysis focuses on infection rates, death counts, global trends, and vaccination progress using real-world pandemic data.

Why this project

  • Practice advanced SQL with real-world public health data.
  • Extract business insights from COVID-19 trends and country comparisons.
  • Combine data cleaning, aggregation, windowing, and visual storytelling.
  • Document results in a simple website to share the analysis.

What you can learn

  • How to analyze pandemic data using SQL.
  • How to compare infection and death statistics across countries.
  • How to link vaccination progress with pandemic severity.
  • How to present findings clearly using charts and summaries.
